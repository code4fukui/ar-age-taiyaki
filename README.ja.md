# ar-age-taiyaki

## age-taiyaki

by [カフェ＆ランチ アリス](https://alice-cafe.business.site/)

## ブログ

https://fukuno.jig.jp/3759

## デモ
このサンプルは、アンカーを使用して仮想オブジェクトを安定した現実世界の場所に配置する方法を示しています。伝統的な日本の和菓子「たい焼き」の3DモデルがAR空間に浮遊する様子を表示します。

## 特徴
- たい焼きの3D表示
- たい焼きの浮遊アニメーション

## 使い方
AR対応デバイスでは、ページを開くとたい焼きが表示されます。
